HTML Introduction CSS Introduction Javascript Introduction

HTML Introduction

  • It looks like base of web development .
  • It is a markup language .
  • HTML stands for Hyper Text Markup Language .
  • It also include other type files like CSS and Javascript .
  • All the html files are the web pages .

CSS Introduction

  • CSS stands for Cascading Style Sheet .
  • CSS file provides Styles and Designing to the HTML Web Page elements .
  • CSS easily make Webpage Responsive to all devices and Screen sizes .
  • CSS is created by World Wide Web Consortium (W3C) to reduce the uses of designing in the HTML file itself .
  • CSS files saves too much time work because It can be saved in other .css file and can be applied on entire website

Javascript Introduction

  • Javascript is Simple but Powerful client side scripting language .
  • Javascript helps to make webpages more interactive .
  • Other than Websites , Javascript is widely used in Game development and Mobile development .
  • Javascript was developed by Brenden eich in 1995 and was used in popular browser Netscape of that time .
  • This language was initially called LiveScript and later renamed JavaScript .
