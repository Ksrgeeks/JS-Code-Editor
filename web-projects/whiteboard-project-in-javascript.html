Develop simple Whiteboard project

In this Exercise , We are going to learn developing a simple Whiteboard project . We can use this Whiteboard project for drawing and Teaching purposes .

Requirements

We need any Text editor like Notepad++ for developing this project .

How to Start ?

We only need two files index.html and whiteboard.js .

Obiviously file names can be changed but .html and .js extensions are important .

index.html file

  • In the index.html file we need an input box to take pencil color code from user and canvas area for drawing .
  • Use input tag for using input box .
  • <input id="hex" placeholder="Enter Hex Color"> </input>
  • To identify this input box we set the id = "hex" .
  • Use canvas tag for using drawing area .
  • <canvas id ="draw"> </canvas>
  • To identify canvas , Use id="draw" .

So finally index.html file looks like :

<!DOCTYPE html> <html lang="en"> <head> <meta charset="utf-8"> <title> Whiteboard </title> <meta name="viewport" content="width=device-width , initial-scale=1"> <script src="app.js"> </script> </head> <input id="hex" placeholder="Enter Hex Color"> </input> <canvas id ="draw"> </canvas> </html>

whiteboard.js file

  • Retrieve the canvas value and store in a variable .
    var canvas = document.getElementById("draw");
  • Use .getContext() to initiate the canvas context and resize() function to set size of window width and height .
    var ctx = canvas.getContext("2d") ; resize() ; function resize() { ctx.canvas.width = window.innerWidth ; ctx.canvas.height = window.innerHeight ; }
  • Let's set the pencil cursor positions inside canvas .
  • Consider the Initial positions of cursor at (0,0) co-ordinates .
    var pos = {x:0 , y:0} ;
  • Now drawing starts from the position , where mouse is clicked . So , let's use setPosition() function to set the new cursor position .
    function setPosition(e) { pos.x = e.clientX ; pos.y = e.clientY ; }
  • Now drawing will start after mouse click so use draw() to check mouse is clicked or not . Retrieve the color code input , if the mouse is clicked .
    Also set some drawing pencil attributes to make drawing intersting . You can change these attributes as you need .
    function draw(e){ if(e.buttons !==1) return ; var color = document.getElementById("hex").value ; ctx.beginPath() ; ctx.lineWidth = 10 ; ctx.lineCap = "round" ; ctx.strokeStyle = color ; ctx.moveTo(pos.x , pos.y) ; setPosition(e) ; ctx.lineTo(pos.x , pos.y) ; ctx.stroke() ; }
  • Now use .addEventListener("command",functionName) function to call any particular function and applying command to perform action .
    window.addEventListener("resize",resize) ; document.addEventListener("mousemove",draw) ; document.addEventListener("mousedown",setPosition) ; document.addEventListener("mouseenter",setPosition) ;

So finally whiteboard.js file looks like :
